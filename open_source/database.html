Graph Databases#

Graph databases are designed to store and manage graph-structured data, which represents entities as nodes and their relationships as edges. These databases are especially useful for applications like social networks, recommendation systems, and knowledge graphs. [1]

  • Neo4j - https://neo4j.com/

  • FalkorDB - FalkorDB/FalkorDB -

  • ArangoDB - arangodb/arangodb -

  • OrientDB - orientechnologies/orientdb -

  • Hugegraph - apache/incubator-hugegraph -

  • SurrealDB - surrealdb/surrealdb -

Semantic Web Standards and Frameworks#

Semantic web standards and frameworks enable the representation, sharing, and linking of data across systems in a structured and machine-readable manner. [1]

  • RDF - Resource Description Framework

  • OWL - Web Ontology Language

  • SKOS - Simple Knowledge Organization System
